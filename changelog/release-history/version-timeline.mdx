# Version Timeline

An at-a-glance chronological index of Linkwarden's major, minor, and patch releases. This timeline helps you track how Linkwarden has evolved by summarizing headline features, important fixes, and providing references for deeper exploration. Use this as your definitive source to follow product progress and prepare for any updates.

---

## Understanding the Version Timeline

Linkwarden's Version Timeline is organized in reverse chronological order, starting with the most recent release. Each entry captures the version number, release date, and a concise summary of key features, improvements, and fixes. This page is designed to provide you with a clear, user-centric view of how each version advances your ability to preserve, organize, and collaborate on web content.

Whether you are a new user reviewing recent enhancements or a long-term user preparing for upgrades, the Version Timeline ensures you have the right information at your fingertips.

## Recent Releases

### Version 1.8.0 — 2024-05-20
- Introduced AI-driven smart tagging enhancements to improve automatic organization of saved links.
- Added new archival format support for monolith preservation, ensuring complete offline snapshots.
- Improved bulk operations with multi-selection UI for faster tag and collection updates.
- Fixed intermittent issues with RSS feed subscriptions failing to poll.
- [See detailed release notes](https://github.com/linkwarden/linkwarden/releases/tag/v1.8.0)

### Version 1.7.5 — 2024-04-15
- Resolved link archive refreshing bug that occasionally prevented updated screenshots.
- Enhanced mobile app authentication stability and session management under low connectivity.
- Minor UI tweaks to the collections sidebar for better visibility.
- Security patch applied to session token refresh mechanism.
- [See detailed release notes](https://github.com/linkwarden/linkwarden/releases/tag/v1.7.5)

### Version 1.7.0 — 2024-03-10
- Added support for customizable dashboard layouts allowing user-specific interface preferences.
- Introduced advanced search filters including date range and tag combinations.
- Optimized preservation worker processes for faster PDF generation.
- Deprecated legacy import formats; users encouraged to update import workflows.
- [See detailed release notes](https://github.com/linkwarden/linkwarden/releases/tag/v1.7.0)

## Historic Release Highlights

### Version 1.6.3 — 2024-01-25
